# 💧 Water Bottling ERP System - Streamlit Edition

A comprehensive Enterprise Resource Planning system designed specifically for water bottling companies, built with Streamlit for a modern web-based interface.

## Features

### ✅ Completed Modules
- 🔐 **Authentication**: Secure login with role-based access control
- 📊 **Dashboard**: Real-time KPIs and analytics
- 👤 **User Management**: User creation, role assignment, and permissions
- 📦 **Product Management**: Product catalog with stock tracking
- 🏭 **Supplier Management**: Supplier database and relationship management
- 👥 **Customer Management**: Customer database and credit management

### 🚧 Coming Soon
- 📦 Inventory Management
- 🛒 Procurement
- 🏭 Production
- 🧪 Quality Control
- 📊 Sales
- 💰 Finance
- 📈 Reports
- ⚙️ Settings

## Technology Stack

- **Frontend**: Streamlit
- **Database**: PostgreSQL
- **Authentication**: bcrypt
- **Data Visualization**: Plotly
- **Data Processing**: Pandas
- **ORM**: SQLAlchemy
